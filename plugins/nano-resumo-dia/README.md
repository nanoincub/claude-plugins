# Nano Resumo Dia — Plugin para Claude Code

Extrai uma timeline de trabalho a partir dos históricos de sessão do Claude Code, formatada como tabela Markdown com horário BRT.

## Instalação

### Via Marketplace Nano Incub (recomendado)

Registre o marketplace (uma vez):

```bash
claude plugin marketplace add nanoincub/claude-plugins
```

Instale o plugin:

```bash
claude plugin install nano-resumo-dia@nano-incub
```

## Uso

Basta pedir naturalmente:

- "o que fiz ontem?"
- "timeline de hoje"
- "resumo do dia 2026-04-02"
- "meu trabalho de ontem"

## Funcionalidades

- Timeline detalhada com horário, projeto, branch, descrição e métricas
- Resumo por projeto com divisão manhã/tarde
- Sugestão de time track com blocos de 15 minutos
- Saída em Markdown ou JSON
